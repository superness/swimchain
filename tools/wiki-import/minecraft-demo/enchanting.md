# Enchanting

![Pixel-art enchanting table with floating book]({{media:enchanting.png}})

**Enchanting** is the system in [[Minecraft]] that adds magical bonuses to tools, weapons, and armor. An enchanted item keeps its normal function but gains extra powers — mining faster, dealing more damage, or protecting the wearer — making enchanting one of the biggest power spikes in the game.

## The enchanting table

The core tool is the **enchanting table**, crafted from obsidian, [[Diamond]], and a book. Using it costs **experience levels**, earned by mining, smelting, defeating mobs, and trading. Placing bookshelves around the table unlocks stronger enchantments, up to a maximum at fifteen shelves — a detail that turns many players' first base into a small library.

Enchantments from the table are partly random: the player is shown a hint but not the full result, which adds a gamble to each attempt.

## Common enchantments

- **Efficiency** — break blocks faster; a staple for any [[Mining]] pickaxe.
- **Fortune** — increases ore and crop drops, especially valuable on a diamond pickaxe.
- **Silk Touch** — harvests blocks in their original form.
- **Sharpness / Protection** — more weapon damage and more armor defense.
- **Unbreaking / Mending** — make gear last far longer, with Mending repairing items using collected experience.

## Anvils and books

Beyond the table, players combine enchantments using an **anvil** and **enchanted books**. A book found in a dungeon, traded from a [[Villager]], or fished up can be applied to gear, letting players plan an exact loadout rather than relying on the table's randomness. High-tier setups often pair enchanted [[Netherite]] gear with Mending so it effectively never wears out.

See also: [[Crafting]] for building the table itself, and [[Villager]] for trading to obtain rare enchanted books.
