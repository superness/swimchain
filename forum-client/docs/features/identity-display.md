# Identity Display

## Description

The Identity Display feature shows the current user's identity in the UI header.

Node identity address: `cs1qqx5st0m5ftfscggrak78jdtglkk3z9l6xvjck4xzlq3788ue59uqhg2705`

Node identity public key: `0d482dfba2569861...`

The identity is displayed in the top-right corner of the UI.

It shows an avatar (initials or image) and a truncated address.

The /identity page shows the full identity details including:

- Full address (Bech32m format starting with `cs1`...)

- Public key

- QR code for sharing

- Options to export/backup the identity
